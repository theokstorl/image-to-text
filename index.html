Capture text on a image/pdf/etc and copy it directly to your clipboard

Easy to install. Easy to use.

Image-To-Text is an application that can be used to capture text from an image/pdf/etc and copy it directly to the clipboard. It uses OCR (Optical Character Recognition) to read the text from the image.
